Hover the map for feature names and attributes. Turn on one or more map layers under the map to stack county boundaries with U.S. House, Texas Legislature, Courts of Appeals (distinct colors per appellate district), Court of Criminal Appeals (statewide), school districts, and water districts (bundled GeoJSON from Census / TCEQ / local dissolve — similar idea to layered plans in tools like Dave’s Redistricting). Click anywhere in Texas (or search an address) to load county-level offices, races, and officials from this database. Address search uses the U.S. Census geocoder.

MapLibre. Counties and (when bundled) overlays: local GeoJSON under static/geo/. Legislative layers: U.S. Census TIGERweb Legislative; school districts: TIGERweb School; water: TCEQ. Courts of Appeals / Court of Criminal Appeals: counties dissolved per Tex. Gov’t Code §22.201 (CCA is statewide final criminal review). Not a substitute for official election maps. Re-run fetch_texas_legislative_geojson if bundled files predate changes. Extra basemaps use CARTO. SBOE boundaries: Texas Capitol Data PLAN E2106 (shapefile → local GeoJSON). Places / CDPs / urban areas: Census TIGERweb Places and Urban. Layer combinations can be shared via the URL hash (updated when you toggle layers).
