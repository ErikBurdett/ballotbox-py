## Security policy

### Supported versions

This repository is currently supported on the `main` branch.

### Reporting a vulnerability

Please do **not** open public issues for suspected security vulnerabilities.

- Email: `security@yourdomain.example` (replace with your real address before publishing)
- Include: description, impact, reproduction steps, affected versions/commits, and any suggested fix

We’ll acknowledge receipt and work on a fix, then coordinate disclosure.
